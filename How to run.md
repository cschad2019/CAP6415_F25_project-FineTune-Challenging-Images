# How to Run the Project

This document explains how to set up the environment and run both the **baseline** and **fine-tuned** experiments for:

> *Fine-tuning a Small Model for Challenging Test Images* (CAP6415 F25, Project #8)

The project is designed to run **on CPU only** and has been tested on Windows with Python 3.12 (adjust as needed).

---

## 1. Prerequisites

- Python 3.11+ (tested with 3.12)  
- Git  
- Enough disk space for CIFAR-10 (~170 MB)  
- (Optional but recommended) VS Code with the Python extension
